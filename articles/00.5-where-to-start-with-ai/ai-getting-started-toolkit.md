# Python Muse -- AI Getting Started Toolkit

**A Practical Checklist for Finance & Accounting Professionals**

---

## STEP 1: Build Your AI Foundation (Mindset First)

- [ ] Read *Scary Smart: The Future of Artificial Intelligence* by Mo Gawdat
- [ ] Reflect: How does AI actually learn?
- [ ] Identify 3 ethical considerations relevant to your role
- [ ] Write down 3 ways AI could improve your current workflow
- [ ] Commit to experimenting -- not mastering -- in the first 30 days

**Goal:** Understand AI before trying to optimize with it.

---

## STEP 2: Install the Core Tools

- [ ] Download and install [Visual Studio Code](https://code.visualstudio.com/)
- [ ] Open it (yes, that counts)
- [ ] Explore the folder structure view
- [ ] Create your first project folder
- [ ] Install the Python extension inside VS Code

**Goal:** Get comfortable navigating your files in a structured way.

---

## STEP 3: Install Python

- [ ] Download Python from [python.org](https://www.python.org/downloads/)
- [ ] During installation, check "Add Python to PATH"
- [ ] Verify installation using Command Prompt (`python --version`)
- [ ] Connect Python interpreter inside VS Code
- [ ] Create and run your first simple `.py` file

Example first file:

```python
print("Hello, AI world.")
```

**Goal:** Remove fear around "developer tools."

---

## STEP 4: Connect AI to Your Workflow

- [ ] Activate Microsoft Copilot (if available in your organization)
- [ ] Or subscribe to a productivity AI tool you'll actually use
- [ ] Practice rewriting one email using AI
- [ ] Improve one Excel formula using AI assistance
- [ ] Document one process with AI help

**Goal:** Move from curiosity to daily usage.

---

## STEP 5: First Automation Project (Keep It Small)

Choose ONE:

- [ ] Automate a repetitive Excel cleanup task
- [ ] Generate a recurring financial report using Python
- [ ] Create a simple script that renames files in bulk
- [ ] Build a basic CSV data summary
- [ ] Draft a reusable AI prompt library for your team

**Rule:** Start small. Finish something.

---

## 30-Day Commitment Plan

| Week | Focus |
|------|-------|
| **Week 1** | Read + Install |
| **Week 2** | Practice Python basics |
| **Week 3** | Connect AI tools to daily work |
| **Week 4** | Complete one automation |

---

## Common Myths (Debunked)

| Myth | Reality |
|------|---------|
| "This is only for developers." | AI amplifies professionals who understand how to use it. |
| "I'm too late to start." | Structure beats hype. |
| "AI will replace me." | Small experiments beat big intentions. |

- What task wastes the most time in your current role?

---

## Final Reminder

You don't need to become a programmer.

You need to become:

- **Comfortable** with tools
- **Curious** about automation
- **Intentional** about ethics
- **Willing** to experiment

That's it.
