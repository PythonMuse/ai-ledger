## When to Use This Skill

Use this skill when:

- Reviewing a transaction or filing for [topic] applicability
- Comparing current documentation against checklist requirements
- Checking whether source guidance has changed

---

## Rules

- Always separate official guidance from company interpretation.
- Never make a final [tax/compliance/accounting] determination.
- Flag items that require [tax manager / reviewer] input.
- Use the company documentation checklist before recommending eligibility or treatment.
- If source guidance has changed since the last review, stop and request human review before applying the guidance.
- Do not invent requirements that are not in the source or company guidance files.

## Output Format

When assisting with a review, provide:

1. Summary of findings
2. Documentation gaps (if any)
3. Questions requiring tax manager input (if any)
4. Recommendation (preliminary only — not a final determination)
